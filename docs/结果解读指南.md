# 结果解读指南

## 综合强势榜

综合强势榜由多周期收益、近1年回撤、加速分和过热惩罚构成。它适合发现观察对象，不适合直接作为买入清单。

## 精选观察池

精选观察池是综合分靠前且 A/C 去重后的少量基金。应重点看：

- 是否由同一主题驱动；
- 是否近1年收益过高；
- 是否回撤已经扩大；
- 是否连续几期留存。

精选观察池必须保持纯评分逻辑：不限制基金公司、不限制基金经理、不限制同一主题。若某个基金公司或基金经理近期确实最强，这本身就是重要信号，不能强行打散。

## 分散观察池

分散观察池是辅助观察视角，不替代精选观察池。它从精选池或实用观察池候选中继续挑选，尽量保留高分基金，同时控制：

- 同一基金公司最多 2 只；
- 若能识别基金经理，同一基金经理最多 2 只。

精选观察池反映最强信号，可能高度集中；分散观察池用于降低重复暴露，便于人工观察。

## 短期异动雷达

短期异动雷达是 V1.1 新增观察层，不替代 V1 精选观察池、分散观察池、主线判断或中线评分。V1 精选观察池适合中线趋势观察；短期异动雷达用于发现最近几日、近1周或近1月刚开始变强的基金和主题。

阅读短期榜单时应同时看：

- 近1周、近5日、近10日收益是否同步靠前；
- 近1月是否开始冒头，但近1年是否已经极端过热；
- 是否首次进入短期异动 Top50 或观察池；
- 是否只是老热门继续冲高；
- 是否出现新的主题扩散苗头；
- 回撤和热度提示是否已经明显恶化。

短期异动不是预测明天涨幅。短期榜单波动更大、误报更多，不能直接当买入推荐，只能作为信息发现和人工复盘入口。`短期异动验证` 只检查历史扫描日之后已到期窗口是否跑赢全候选池平均，未到期窗口不参与均值、胜率和评价。

## 主线预警

主线判断规则：

- 核心主线：主题覆盖率 >= 80%，且合计持仓占比 >= 100；
- 强主线：主题覆盖率 >= 60%，且合计持仓占比 >= 50；
- 次强方向：主题覆盖率 >= 40%，且合计持仓占比 >= 20；
- 否则为分散/弱方向。

状态判断会进一步提示“明显偏热”“仍可持续跟踪”“观察是否扩散”“疑似退潮”等。

## 时间机器报告

重点看：

- 当时系统是否识别出后来强势的方向；
- 识别依赖的是短期加速、中期趋势、长期趋势还是低回撤；
- 未来收益是否跑赢全候选池；
- 样本内有效的规则是否在留出区间失效。

`组合后验表现支持` 的含义：

- 这不是未来持仓验证；
- 这不是主题本身的独立收益验证；
- 它只是说明：当时识别出该主题的精选组合，在后续已到期窗口中是否跑赢全候选池或基准；
- 因此只能作为后验支持信号，不能当作严格主题因子验证。
