# 使用说明

## 日常使用流程

每周运行：

```bash
python scripts/run_market_scan.py --limit 800
python scripts/run_tracking.py
```

想看短期冒头基金：

```bash
python scripts/run_short_term_radar.py --limit 800
```

想复盘某个历史日期：

```bash
python scripts/run_time_machine.py --as-of YYYY-MM-DD --horizon 1m,3m,6m,12m --top-n 10 --limit 800
```

想验证新星、生命周期、资金迁移信号：

```bash
python scripts/run_v2_lite_validation.py --limit 800
```

主要输出位置：

- `reports/<日期>/基金雷达扫描结果.xlsx`
- `reports/<日期>/分析摘要.md`
- `reports/<日期>/短期异动雷达.xlsx`
- `reports/v2_lite/<日期>/V2验证报告.xlsx`

## 阶段收益对比

```bash
python scripts/run_stage_return_compare.py
```

默认读取 `config/fund_list.yaml`。也可以传入代码：

```bash
python scripts/run_stage_return_compare.py --codes 025422,021528,011452
```

## 全市场扫描

```bash
python scripts/run_market_scan.py --limit 800
```

首次运行会较慢，因为需要拉取并缓存净值。

## 时间机器

```bash
python scripts/run_time_machine.py --as-of 2025-06-18 --horizon 1m,3m,6m,12m --top-n 10
```

## 短期异动雷达

```bash
python scripts/run_short_term_radar.py --limit 800
```

可选参数：

```bash
python scripts/run_short_term_radar.py --limit 800 --top-n 50 --lookback 3d,5d,10d,1w,1m --verify-horizon 1w,2w,1m,3m
```

该模块只作为新增观察层，用于发现近期刚变强的基金和主题，不改变 V1 精选观察池、分散观察池和主线判断。

## 短期权重搜索

```bash
python scripts/run_short_term_weight_search.py
```

## 回测

```bash
python scripts/run_backtest.py --start 2023-01-01 --top-n 10
```

## 权重搜索

```bash
python scripts/run_weight_search.py --train-start 2023-01-01 --train-end 2025-06-18 --valid-start 2025-06-18
```
