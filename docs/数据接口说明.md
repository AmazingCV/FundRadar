# 数据接口说明

主要数据源是 AKShare：

- `fund_name_em`：基金代码、名称、类型。
- `fund_open_fund_rank_em`：开放式基金排行，用于快速粗筛。
- `fund_open_fund_info_em`：基金累计净值或单位净值走势。
- `fund_portfolio_hold_em`：基金持仓。
- `stock_zh_index_daily`：沪深300等指数行情，用于时间机器基准。

接口保护：

- 每个接口有重试；
- 净值、持仓、排行表写入 `data/cache/`；
- 字段名使用候选列表自动识别；
- 单只基金失败写入失败表，不中断主流程；
- AKShare 不可用时会明确报错，要求安装依赖。
