# 系统设计

## 模块

- `data_loader.py`：AKShare 数据接口、缓存、字段兼容、重试。
- `fund_filter.py`：主动权益基金过滤、A/C 去重、观察池强制加入。
- `metrics.py`：阶段收益、未来收益、最大回撤、波动率。
- `scoring.py`：可配置评分、加速分、过热惩罚、热度提示。
- `diversification.py`：基金公司/基金经理识别和分散观察池生成。
- `pipeline.py`：阶段收益对比和全市场扫描主流程。
- `holdings.py`：精选基金重仓股抓取与统计。
- `theme_analysis.py`：主题标签、主线强度和状态判断。
- `backtest.py`：月度历史回测。
- `time_machine.py`：历史时点模拟和后验验证。
- `weight_search.py`：权重方案搜索、训练/验证切分。
- `short_term_radar.py`：V1.1 短期异动雷达、新星基金榜、短期主题异动和轻量历史验证。
- `tracking.py`：快照保存和上期对比。
- `report.py`：Excel、Markdown、图表输出。

## 数据流

1. 拉取基金基础信息或排行表；
2. 过滤主动权益基金，合并观察池；
3. 拉取净值并缓存；
4. 在 as-of 日期截断净值；
5. 计算收益、回撤、波动率和评分；
6. 输出榜单和观察池；
7. 保留纯评分精选观察池，同时生成分散观察池；
8. 抓精选池持仓，做主题归因；
9. 保存报告和快照；
10. 回测或时间机器使用未来真实净值验证。
11. 短期异动雷达复用候选池、净值缓存和主题归因，在独立评分层计算近1日到近1月的短期强度、排名提升、过热惩罚和新进状态，并单独输出短期报告。
