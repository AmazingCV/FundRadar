## v4.2-daily-runner

- 新增 `src/fund_radar/daily_runner/` 和 `scripts/run_daily_all.py`，作为日常统一调度入口。
- 新增 step-level run guard，状态写入 `reports/daily/YYYY-MM-DD/run_guard.json`。
- 默认跳过已完成、输出存在且最新净值日期未变化的步骤，避免每天重复无意义重算。
- 支持 `--status` 查看状态、`--step` 单步执行、`--report-only` 只重建晨报、`--force` 强制重跑。
- 每一步输出 `[START]`、`[DONE]`、`[SKIP]`、`[FAIL]` 和耗时。
- V4.1 晨报展示层继续保留 6 位基金代码，并对 A/C 等不同份额做合并展示；底层原始报告不改。
